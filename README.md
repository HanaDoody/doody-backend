# Doody Backend

## 프로젝트 소개

Doody Backend는 사용자의 일상 회복 루틴을 돕는 Spring Boot 기반 백엔드 서버입니다.  
아침/저녁 리듬 기록, 개인화 미션 추천, AI 채팅 응답, 리포트 생성, 두디 컬렉션 관리, 하나 금융 혜택 해금 기능을 제공합니다.

사용자의 리듬, 자율성, 연결감 데이터를 기반으로 작은 행동을 꾸준히 이어갈 수 있도록 기록과 보상 흐름을 설계했습니다.

## 기술 스택

| 구분 | 기술 |
| --- | --- |
| Language | Java 21 |
| Framework | Spring Boot 4.0.6 |
| Build Tool | Gradle |
| Database | PostgreSQL |
| ORM | Spring Data JPA, Hibernate |
| Cache / Infra | Redis |
| API Docs | Springdoc OpenAPI |
| Utility | Lombok |

## 주요 기능

- 회원가입 및 로그인
- 온보딩 기반 초기 목표 설정
- 아침/저녁 리듬 체크
- 사용자 상태 기반 오늘의 미션 추천
- 자율성 미션 진행, 거절, 인증, 완료
- AI 채팅 메시지 응답
- 회복 리포트 조회
- 두디 컬렉션 및 지도 기반 컬렉션 캡처
- 하나 금융 혜택 해금 상태 관리
